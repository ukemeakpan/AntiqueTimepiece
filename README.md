# AntiqueTimepiece Registry

A specialized blockchain platform for authenticating vintage watches and clocks, providing comprehensive provenance tracking and authentication for antique timepiece collectors.

## Features

- Timepiece authentication and registration
- Manufacture year validation
- Maker provenance tracking
- Mechanism classification
- Auction status management

## Smart Contract Functions

### Public Functions
- `register-timepiece`: Register an antique timepiece
- `auction-timepiece`: Update timepiece auction status

### Read-Only Functions
- `get-timepiece-data`: Retrieve timepiece information
- `get-curator`: Get curator's address

## Supported Timepiece Types
- Pocket Watch, Wristwatch, Mantel Clock, Wall Clock, Grandfather Clock, Marine Chronometer

## Mechanism Types
- Mechanical, Automatic, Spring-driven, Pendulum, Quartz
